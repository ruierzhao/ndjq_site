### 更改邮箱以及密码
客户端检查邮箱和密码是否一致
1、邮箱一致
    - 请求 > /i/modifyEmail 
    - 服务器判断验证码是否一致
        - 如果一致，更新此数据，找到密码更新
        - 不一致， 错误码返回客户端
2、邮箱不一致
    - 请求服务器，判断邮箱是否已存在
        - 存在，错误码返回客户端
        - 不存在，拿到nickname 更新数据库（req.session.user）
